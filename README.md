# Ultimate Edge & Corner Detector

Compare **Sobel**, **Canny**, **Harris**, and **Shi-Tomasi** algorithms interactively.

## Features
- Upload an image
- Adjust parameters with sliders
- Compare results side-by-side
- Save processed images with one click

## Run Locally
```bash
git clone <repo_url>
cd edge_corner_detector
pip install -r requirements.txt
streamlit run app.py
